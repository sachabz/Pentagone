# Pentagone

A financial website with a cashflow calculator

## Environement

Vue 3
Typescript
Pinia store
ElementPlus

## Made by

Sacha Zwertvaegher

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
